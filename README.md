# Zionic
- [x] Ionic 1.3
- [x] Typescript
- [x] ngCordova (Geolocation)
- [x] http & promises (weather & flickr apis)
- [x] Jasmine, Karma & Protractor
- [x] Webpack
- [x] Crosswalk


## Installation

to run

```bash
$ npm run prep
$ npm run watch
$ npm start
```

to test

```bash
$ npm test
```
